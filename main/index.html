Eclipse S-CORE

Welcome & Overview#

Welcome to the Software documentation of the SCORE project.

If this is the first time you get in touch with SCORE, the SCORE main page may be a good starting point to receive first insights of the project itself.

Introduction#

Scope

The project develops an open source core stack for Software Defined Vehicles (SDVs), targeting embedded high-performance Electronic Control Units (ECUs). Check our Scope for details.

Project Status

Check out our roadmap to see the projectâ€™s key milestones and features we have planned. Click here to explore!

How to Contribute?

The only way to influence Eclipse SCORE is to contribute - SCORE is open! Please check our Get Involved guide for more information.
